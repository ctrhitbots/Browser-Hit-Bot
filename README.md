## 🚀 Getting Started · Başlangıç

</div>

### Prerequisites · Gereksinimler

- Go 1.21+
- Google Chrome or Chromium
- Windows, macOS, or Linux

### Build · Derleme

```bash
# Clone the repository · Repoyu klonla
git clone https://github.com/YOUR_USERNAME/ctr-hit-bot
cd ctr-hit-bot

# GUI Application (Desktop) · GUI Uygulaması (Masaüstü)
cd serp-gui
wails build

# CLI Tool (Server/VPS) · CLI Aracı (Sunucu/VPS)
cd go-serp-scraper
go build ./cmd/...
```

### Configuration · Yapılandırma

```json
{
  "keywords": ["best seo tool 2025", "ctr optimization software"],
  "target_domains": ["yourdomain.com"],
  "headless": false,
  "language": "en",
  "max_results": 10,
  "platform": {
    "desktop_ratio": 60,
    "mobile_ratio": 40,
    "desktop_os": [
      { "name": "Windows", "ratio": 70 },
      { "name": "macOS",   "ratio": 30 }
    ],
    "mobile_os": [
      { "name": "Android", "ratio": 60 },
      { "name": "iOS",     "ratio": 40 }
    ]
  },
  "mouse_movement": { "enabled": true, "idle_movements": true },
  "text_selection": { "enabled": true, "probability": 70 },
  "multi_click":    { "enabled": true, "probability": 50, "min_wrong_clicks": 1, "max_wrong_clicks": 3 },
  "tab_switching":  { "enabled": true, "probability": 80 },
  "real_engagement":{ "enabled": true, "play_video": true, "form_hover": true },
  "pagination":     { "enabled": true, "max_pages": 3, "scroll_each_page": true },
  "delays": {
    "after_navigate_min_ms": 1500,
    "after_navigate_max_ms": 3500,
    "after_click_min_ms": 5000,
    "after_click_max_ms": 12000,
    "between_keywords_min_ms": 8000,
    "between_keywords_max_ms": 20000
  }
}
```

```bash
# Run all keywords · Tüm keywordleri çalıştır
./serp-scraper -config config.json

# Single keyword · Tek keyword
./serp-scraper -config config.json -q "best seo tool 2025"
```
